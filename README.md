Juego de cartas en Java.

Este programa es una simulación de cómo funciona un mazo de cartas real en la computadora. 
Fue creado para la clase de Programación y sirve para demostrar cómo organizar información 
(como valores y figuras de cartas) de forma ordenada.

El programa puede generar las 52 cartas completas, revuelve las cartas para que salgan al azar, 
entrega una cantidad de cartas a diferentes jugadores, permite que un jugador regrese cartas al 
mazo y lleva la cuenta de cuántas quedan disponibles.

Para hacerlo funcionar seria necesario descargar la carpeta y abrirlos en su aplicación para
programar de confianza.

Cuando el programa corra, verá una serie de mensajes en pantalla como los siguientes.
    Primero, te avisa que el mazo de 52 cartas ya está listo.
    Luego, verá un mensaje diciendo que se están revolviendo las cartas.
    Después, aparecerán los nombres de los jugadores y las cartas que le tocaron a cada uno.
    Finalmente, mirará un ejemplo de un jugador devolviendo una carta y el mazo actualizándose automáticamente.
